# Программа сопоставления товаров

Программа для автоматического сопоставления товаров из разных каталогов по названиям с использованием алгоритмов нечеткого поиска.

## Установка

1. Убедитесь что у вас установлен Python 3.7+
2. Запустите `install_requirements.bat` для установки зависимостей
3. Или установите вручную: `pip install -r requirements.txt`

## Запуск

Запустите файл `product_matcher_gui.py`:
```
python product_matcher_gui.py
```

## Использование

1. **Выберите файлы**: 
   - Файл каталога сайта (Excel с колонками ID и наименование)
   - Файл программы учета (Excel с колонками ID и наименование)

2. **Настройте порог схожести**: 
   - 60-70% - для строгого сопоставления
   - 40-60% - для более мягкого поиска совпадений

3. **Запустите сопоставление**: Нажмите кнопку "Начать сопоставление"

4. **Результат**: Программа создаст файл `результат_сопоставления.xlsx` с найденными парами

## Формат входных файлов

### Файл сайта
- Должен содержать колонки с ID товара и наименованием
- Поддерживаемые названия колонок: ID, _ID_, код, артикул (для ID)
- Наименование, название, товар, name (для названий)

### Файл программы учета  
- Аналогичная структура с ID и наименованием товаров

## Алгоритм работы

1. **Очистка названий**: Удаление спецсимволов, транслитерация
2. **Нечеткий поиск**: Использование алгоритма WRatio для поиска похожих названий
3. **Фильтрация**: Отбор совпадений выше заданного порога
4. **Экспорт**: Сохранение результатов в Excel

## Особенности

- Автоматическое определение колонок в файлах
- Обработка русских и английских названий
- Удаление указаний стран (/Польша/, /Германия/)
- Прогресс выполнения в реальном времени
- Настраиваемый порог схожести
