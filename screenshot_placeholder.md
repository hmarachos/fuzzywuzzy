# Скриншот программы

Для добавления скриншота в README.md:

1. Сделайте скриншот работающей программы
2. Сохраните его как `screenshot.png` в корне проекта
3. Скриншот автоматически отобразится в README.md

Рекомендуемый размер: 800x600 пикселей

## Что должно быть на скриншоте:
- Главное окно программы
- Выбранные файлы
- Настройки порога схожести
- Прогресс-бар в процессе работы
- Статистика обработки
